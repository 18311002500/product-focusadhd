```
FocusFlow is a productivity tool specifically designed for users with ADHD (Attention Deficit Hyperactivity Disorder). Addressing the pain point that traditional task management tools are "ineffective" for the ADHD brain, we help you build sustainable focus habits through gamification, instant positive feedback, and flexible time management.

【Key Features】

🎮 Gamified Task System
- Task difficulty levels (Easy/Medium/Hard) with corresponding point rewards
- Task pool design: no forced same-day completion, reducing psychological pressure
- Point system: Earn energy points for completing tasks, continuous motivation

⏱️ ADHD-Friendly Focus Timer
- Flexible durations: 5/10/15/20 minutes, better than traditional Pomodoro for ADHD brains
- Pause-friendly: Pausing doesn't count as failure, reducing frustration
- Colorful progress ring: Visual representation of time passing

🏆 Badge Achievement System
- 20+ unique badges for consecutive completions, first achievements, etc.
- Positive reinforcement mechanism, making task completion rewarding
- Badge wall display, visualizing sense of achievement

🌱 Plant Growth Mechanism
- Virtual plant grows, blooms, and bears fruit as points accumulate
- Nurturing elements enhance user engagement
- Unique visual effects at each stage

📊 Detailed Statistics
- Daily/weekly/monthly completion rate tracking
- Focus time statistics
- Consecutive check-in records
- Visualized progress tracking

💪 Health Integration

- Activity reminders: Sedentary reminders
- Exercise badges: Encourage physical activity
```
