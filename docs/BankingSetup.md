# App Store Connect 收款账户配置指南

> 本指南说明如何在 App Store Connect 中配置收款银行账户
> 前置条件：Apple Developer 账号已审核通过
> 预计耗时：15-20 分钟

---

## 🎯 目标

配置收款信息，确保应用收入可以提现到您的银行账户：
- ✅ 签署付费应用协议
- ✅ 完成税务表格（W-8BEN）
- ✅ 添加中国大陆银行账户

---

## 📋 前置条件

- [x] Apple Developer 账号已审核通过
- [x] 已登录 App Store Connect
- [x] 准备以下材料：
  - 身份证
  - 银行卡（支持接收外币）
  - 银行 SWIFT 代码

---

## 🚀 操作步骤

### 步骤 1: 登录 App Store Connect

1. 访问：
   ```
   https://appstoreconnect.apple.com
   ```

2. 使用 Apple Developer 账号登录

---

### 步骤 2: 进入协议、税务和银行业务页面

1. 点击顶部菜单 **"协议、税务和银行业务"**
   (Agreements, Tax, and Banking)

   或直接访问：
   ```
   https://appstoreconnect.apple.com/agreements
   ```

2. 页面显示三个标签页：
   - **协议** (Agreements)
   - **税务** (Tax Forms)
   - **银行账户** (Bank Accounts)

---

## 📄 第一部分：签署付费应用协议

### 步骤 3: 请求付费应用协议

1. 在 **"协议"** 标签页中
2. 找到 **"付费应用协议"** (Paid Applications Agreement)
3. 点击 **"请求"** (Request) 按钮

### 步骤 4: 查看并同意协议

1. 阅读协议内容（可以滚动快速浏览）
2. 勾选 **"我已阅读并同意以上条款"**
3. 点击 **"同意"** (Agree)

### 步骤 5: 确认协议状态

- 协议状态变为 **"已生效"** (Active)
- 这一步完成 ✅

---

## 📝 第二部分：填写税务表格

### 步骤 6: 设置税务信息

1. 点击 **"税务"** (Tax Forms) 标签页
2. 点击 **"设置"** (Set Up) 或 **"添加税务表格"**

### 步骤 7: 选择税务表格类型

**选择：**
```
☑️ U.S. Tax Forms (美国税务表格)
```

### 步骤 8: 填写 W-8BEN 表格

W-8BEN 表格适用于**非美国税务居民**。

#### 第 1 部分：身份信息

**姓名 (Name):**
```
姓氏 (Last Name): [您的姓氏拼音，如 ZHANG]
名字 (First Name): [您的名字拼音，如 SAN]
```

**国籍 (Country of Citizenship):**
```
China (中国)
```

**永久居住地址 (Permanent Residence Address):**
```
国家: China (中国)
地址行 1: [您的详细地址，英文或拼音]
城市: [城市拼音，如 Shanghai]
省份: [省份拼音，如 Shanghai]
邮编: [您的邮编]
```

**邮寄地址 (Mailing Address):**
```
☑️ 与永久居住地址相同
```

#### 第 2 部分：税务识别号

**美国纳税人识别号 (U.S. TIN):**
```
☐ 我没有美国纳税人识别号
```

**外国纳税人识别号 (Foreign TIN):**
```
☑️ 我有外国纳税人识别号
号码类型: China Resident Identity Number (中国居民身份证号码)
号码: [您的18位身份证号码]
```

#### 第 3 部分：声明

**税务协定优惠 (Tax Treaty Benefits):**
```
☑️ 我申请享受美中税收协定优惠
条款: Article 14 (独立个人劳务)
国家: China (中国)
```

**说明：** 这一步可以减少预扣税，建议勾选。

#### 第 4 部分：签名

**签名 (Signature):**
```
输入您的姓名拼音作为电子签名
如: ZHANG SAN
```

**日期 (Date):**
```
自动填充当前日期
```

**容量 (Capacity):**
```
选择: Individual (个人)
```

### 步骤 9: 提交税务表格

1. 仔细检查所有信息
2. 点击 **"提交"** (Submit)
3. 等待审核（通常即时通过）

---

## 🏦 第三部分：添加银行账户

### 步骤 10: 添加银行账户

1. 点击 **"银行账户"** (Bank Accounts) 标签页
2. 点击 **"添加银行账户"** (Add Bank Account)

### 步骤 11: 选择国家/地区

```
国家/地区: China (中国)
```

### 步骤 12: 填写银行信息

**账户持有人姓名 (Account Holder Name):**
```
[您的真实姓名，与银行卡一致，如 ZHANG SAN]
```

**账户持有人类型:**
```
☑️ 个人 (Individual)
```

**银行选择方式:**

**方法 A: 通过 SWIFT 代码搜索（推荐）**
```
SWIFT 代码: [您的银行 SWIFT 代码]
```

常见银行 SWIFT 代码：
| 银行 | SWIFT 代码 |
|------|-----------|
| 中国银行 | BKCHCNBJ |
| 工商银行 | ICBKCNBJ |
| 建设银行 | PCBCCNBJ |
| 农业银行 | ABOCCNBJ |
| 招商银行 | CMBCCNBS |
| 交通银行 | COMMCNSH |
| 浦发银行 | SPDBCNSH |

**方法 B: 通过银行名称搜索**
```
输入银行名称关键词，如 "China Merchants Bank"
```

### 步骤 13: 填写账户信息

**银行账号 (Account Number):**
```
[您的银行卡号]
```

**账户类型:**
```
☑️ 储蓄账户 (Savings)
或
☑️ 支票账户 (Checking)
```

**币种:**
```
CNY (人民币)
```

**银行地址:**
```
通常会自动填充，如不正确手动修改
```

### 步骤 14: 确认账户信息

1. 仔细核对所有信息
2. 确认账户持有人姓名与银行记录一致
3. 点击 **"下一步"** (Next)

### 步骤 15: 验证银行账户

Apple 可能会进行小额验证：
- 向您的账户汇入小额资金（如 $0.01）
- 您需要在银行账单中查看确认码
- 在 App Store Connect 中输入确认码

**注意：** 中国大陆银行账户可能不需要此步骤。

## 📊 收入结算说明

### 结算周期
- **每月结算**：Apple 每月统计收入
- **付款时间**：通常在次月中下旬付款
- **最低付款额**：$10 USD（达到 $10 才会打款）

### 付款方式
- **电汇 (Wire Transfer)**：直接汇入您的银行账户
- **币种**：美元 (USD)，银行自动兑换为人民币

### 费用说明
- **Apple 佣金**：30%（首年），15%（订阅第二年起）
- **银行手续费**：由接收银行收取（约 $15-25/笔）
- **汇率损失**：按银行实时汇率换算

### 收入预估示例
```
应用售价: $4.99
Apple 抽成 (30%): $1.50
开发者收入: $3.49

如每月 100 次付费下载:
月收入: 100 × $3.49 = $349 USD
换算人民币: ≈ ¥2,500（按汇率 7.2）
```

## 🆘 常见问题

### Q1: 可以使用支付宝或微信收款吗？
**解答：** 不可以。Apple 只支持银行账户电汇，不支持第三方支付平台。

### Q2: 需要使用特定的银行卡吗？
**解答：** 建议使用大型银行的借记卡或储蓄卡：
- ✅ 中国银行、工商银行、建设银行等
- ✅ 支持接收外币的账户
- ❌ 不建议使用地方小银行（可能无法接收国际电汇）

### Q3: SWIFT 代码是什么？如何查询？
**解答：**
- SWIFT 代码是银行的国际识别码
- 查询方式：
  1. 致电银行客服询问
  2. 在银行官网查询
  3. 查看银行提供的国际汇款信息

### Q4: 税务表格填写错误怎么办？
**解答：**
- 可以重新提交新的税务表格
- 旧表格会被新表格覆盖
- 建议在每年年初检查和更新

### Q5: 收入如何报税？
**解答：**
- 中国税务居民需要申报境外收入
- 建议咨询专业税务顾问
- 保留所有收入记录和银行流水

### Q6: 付款一直未到怎么办？
**解决：**
- 确认已达到最低付款额（$10）
- 检查银行账户信息是否正确
- 联系银行确认是否有国际汇款
- 联系 Apple 开发者支持
