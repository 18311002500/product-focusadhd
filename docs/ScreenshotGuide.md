## 🎯 截图场景规划

按以下顺序准备 5 张截图，讲述一个完整的故事：

### 截图 1: 今日任务首页
**目的**: 展示核心功能 - 游戏化任务管理

**操作步骤**:
1. 打开 FocusFlow 应用
2. 确保首页有 3-5 个任务（不同难度）
3. 确保植物成长卡片可见
4. 截图

**预期效果**:
- 显示 "今日任务" 标题
- 展示植物成长进度条
- 任务列表清晰可见
- 大按钮、鲜艳色彩

---

### 截图 2: 专注计时器
**目的**: 展示 ADHD 友好的专注工具

**操作步骤**:
1. 点击底部 "专注" Tab
2. 选择一个时长（如 15 分钟）
3. 点击 "开始专注"
4. 等待几秒让进度环显示一点进度
5. 截图

**预期效果**:
- 彩色圆形进度环
- 大号时间显示
- "专注中..." 状态
- 控制按钮可见

---

### 截图 3: 徽章墙
**目的**: 展示成就系统，增强游戏化吸引力

**操作步骤**:
1. 完成几个任务（解锁一些徽章）
2. 点击底部 "徽章" Tab
3. 截图展示已解锁和未解锁徽章

**预期效果**:
- 徽章网格布局
- 已解锁徽章高亮显示
- 未解锁徽章灰色显示
- 徽章名称和描述可见

---

### 截图 4: 进度统计
**目的**: 展示数据追踪和可视化进步

**操作步骤**:
1. 点击底部 "进度" Tab
2. 如果有数据则直接截图
3. 如果没有数据，先完成几个任务再截图

**预期效果**:
- 周/月完成率图表
- 总积分显示
- 连续打卡天数
- 植物成长阶段

---

### 截图 5: 添加任务
**目的**: 展示简洁的任务创建流程

**操作步骤**:
1. 在首页点击 "+" 按钮
2. 填写一个示例任务（如 "完成项目报告"）
3. 选择难度为 "困难"
4. 截图显示添加任务界面

**预期效果**:
- 任务输入框
- 难度选择器
- 直观的界面设计
- 大按钮易于点击

## 🛠️ 截图操作步骤

### 使用 Xcode 模拟器截图

**Step 1: 打开 Xcode 项目**
```bash
cd /path/to/product-focusadhd
open FocusFlow.xcodeproj
```

**Step 2: 选择模拟器**
1. 在 Xcode 顶部工具栏找到设备选择器
2. 选择 **"iPhone 15 Pro Max"** 或 **"iPad Pro (12.9-inch)"**

**Step 3: 运行应用**
1. 点击运行按钮（▶）或按 Cmd+R
2. 等待模拟器启动并加载应用

**Step 4: 截图**
方法 1 - 使用快捷键：
```
Cmd + S          # 保存截图到桌面
```

方法 2 - 使用菜单：
```
Device → Screenshot  # 保存到桌面
```

方法 3 - 使用 Preview 编辑：
```
截图后会在右下角显示缩略图，点击可编辑
```

**Step 5: 重命名截图**
按以下格式重命名：
```
iphone67_screenshot_1_home.png
iphone67_screenshot_2_timer.png
iphone67_screenshot_3_badges.png
iphone67_screenshot_4_progress.png
iphone67_screenshot_5_addtask.png
```

## 🆘 常见问题

### Q1: 模拟器无法启动？
**解决**:
- 确保 Xcode 已完全安装
- 检查模拟器是否在 Xcode → Preferences → Components 中下载
- 重启 Xcode 和模拟器

### Q2: 截图尺寸不对？
**解决**:
- 确保选择了正确的模拟器设备
- 不要缩放模拟器窗口（保持 100%）
- 使用 Cmd+1 确保实际像素尺寸

### Q3: 应用数据如何预设？
**解决**:
- 在模拟器中手动添加一些示例任务
- 完成几个任务解锁徽章
- 数据会保存在模拟器中

### Q4: 截图中有敏感信息？
**解决**:
- 使用预览工具涂抹敏感信息
- 或者重新截图避免敏感内容
